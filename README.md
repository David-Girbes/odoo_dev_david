# Proyecto Odoo

He hecho la gestión de una tienda online de venta de componentes, pórtatiles, sobremesa etc

# Crear una empresa

He creado una empresa

# Modulos instalados o usados

CRM,Ventas,Facturación,Inventario y Sitio web

# Inventario

He definido productos con su precio, stock y he creado categorías para caracterizarlos, y con atributos y variantes especificas para cada producto por ejemplo la Ram es un componente, y tiene de atributo el GB que tiene, he creado varios almacenes

#Sitio web

Puedes registrarte con usuario conntraseña y se creará automáticamente un cliente

Para el sitio web he hecho una web simple de tienda, donde salen los productos que tengo publicados, con varios filtros, por categorías por atributos y opción de mayor a menor precio etc, puedes añadir productos a la cesta para despues comprarlos, habiendo habilitado el método de pago por transferenca bancaria

Cuando compras te crea una factura para descargar, internamente sale en ventas esa nueva compra, la confirmamos, y deja crear una factura para tenerla internamente y nos saldrá en el apartado de facturación y se le puede enviar un coorreo diciendo que ya se ha hecho la compra pasandole la factura y luego y cuando compruebe que hay stock disponible, y se envía a su dirección
